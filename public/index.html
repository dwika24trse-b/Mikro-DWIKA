Panduan Integrasi Mikrokontroler / IoT

Untuk mengirim data dari Arduino, ESP8266/ESP32, atau Raspberry Pi ke database, kirimkan HTTP POST Request dengan format JSON ke endpoint di bawah ini:

POST http://[IP_ADDRESS_SERVER]:5000/api/data
Format JSON Payload: 
{
  "temperature": 25.4,
  "ph": 7.2,
  "turbidity": 95.0,
  "water_level": 82.5
}

Catatan: Kode contoh lengkap untuk ESP32 (Arduino IDE) dan Raspberry Pi (Python) sudah disediakan di folder proyek Anda di sub-folder device-codes/.
